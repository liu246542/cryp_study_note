    Privacy-Preserving Computation

    One-Sided Secure Matrix Multiplication with \ell Colluding Servers

    B\in\mathbb{F}^{n\times p} 是一个公开的矩阵，A\in\mathbb{F}^{m\times n} 是一个私有矩阵，我们的目标是在有最多 \ell 个服务器共谋且不泄露 A 的情况下完成对 AB 的计算。

    假设共谋的服务器集合为 \mathcal{L} = \{i_1, i_2, \ldots, i_\ell\}\subseteq [1:N]，他们获得的部分 A 矩阵为 \~{A}_{\mathcal{L}} \triangleq (\~{A}_{i_1}, \~{A}_{i_2}, \ell, \~{A}_{i_\ell})。

    首先，用户将矩阵 A 垂直拆分成 N-\ell 个子矩阵，即

    A = [A_1 A_2 \ldots A_{N-\ell}]^T

    \forall i, A_i\in\mathbb{F}^{(m/(N-\ell))\times n}，那么 AB 可以写成：

    AB = [A_1B \ldots A_{N-\ell}B]^T

    用户可以按照以下形式进行编码：

    \~{A}_i = \sum_{j=1}^{N-\ell} A_jx_i^{j-1} + \sum_{k=1}^\ell K_kx_i^{k + (N-\ell) -1}

    每个服务器将自己那部分 \~{A}_i 乘以 B，返回的结果为以下形式：

    h(x) = \sum_{j=1}^{N-\ell}A_jBx^{j-1} + \sum_{k=1}^\ellK_kBx^{k+(N - \ell) - 1}

    其中 x=x_i, i=1,\ldots, N。

    这样根据 h(x_i), i=1,\ldots, N 便可以还原出 AB。

    Fully Secure Matrix Multiplication with \ell Colluding Servers

    类似于上一个，除了这次 B 矩阵也是私有的。首先我们选一个 r，将矩阵 A 垂直分成 r 个子矩阵，将矩阵 B 水平分成 r 个子矩阵，即

    A=[A_1 A_2 \ldots A_r]^T,\mbox{ and } B=[B_1 B_2 \ldots B_r]

    其中，A_i\in\mathbb{F}^{(m/r)\times n}，B_i\in\mathbb{F}^{n \times (p/r)}
